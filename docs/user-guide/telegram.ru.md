# Использование в Telegram

Пишите обычным языком; слэш-команды — по желанию.

## Надёжные задачи

| Задача | Пример |
|--------|--------|
| Перевод | «переведи на английский: привет» |
| Счёт | «посчитай 23+45*45» |
| Напоминание | «напомни в 22:50 пошла спать» |
| Погода | «погода в Минске» |
| Новости | «какие новости» (нужен SearXNG) |

## Часовой пояс

`/me` или дефолт `Europe/Moscow`. Можно «в 12:00 utc» в фразе.

## Картинки

- Генерация: «сгенерируй закат» или `/imagine …`
- Фото + подпись в одном сообщении — лучший способ edit
- «Что на фото?» — vision, не генератор

## Обратная связь

👍 / 👎 — ephemeral learning (на проде с одобрением админа).
